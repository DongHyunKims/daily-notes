1. midea 쿼리 : 사이즈에 맞추어 다른효과를 입힐ㄱ때 사용. 반응형 웹사이트에 많이 사용한다.

~~~css
@media screen and (min-width: 480px) {

    body {
        background-color: lightgreen;
    }
}
~~~

2. img , div 등 정가운데 처리

**html**
~~~
<div id="addItem">
	<img src=""/>
</div>
~~~

**css**
~~~css
#addItem > img{
    width: 30%;
    height: 30%;
    position: relative;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    -webkit-transform: translate(-50%, -50%);
}
~~~
